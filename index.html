Learning last week was extremely interesting. The three things I learned mostly about were assignment operators, control flow, and dunder methods. Assignment operators in JS are used to assign variables to values and perform arithmetic (or bitwise) operations on them. Control flow is how you manage the loops and conditionals in your code. Finally, dunder methods are special methods in python that allow you to define how objects of a class behave when they are called. Essentially replacing the default memory location return value with human readable text.
